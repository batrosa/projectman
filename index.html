Помощь

О приложении

ProjectMan — система управления задачами с двойным контролем выполнения. Приложение позволяет создавать организации, проекты, назначать задачи сотрудникам и отслеживать их выполнение в реальном времени.

Организации

Каждая команда работает в своей организации — изолированном рабочем пространстве.

  • Создание: Зарегистрируйтесь и создайте организацию — вы станете её владельцем
  • Присоединение: Получите код приглашения от администратора и введите его при входе
  • Код приглашения: 6-значный код для приглашения сотрудников (виден в меню организации)
Поделитесь ссылкой с кодом приглашения — сотрудник автоматически присоединится после регистрации.

Роли пользователей

Владелец:

  • Полный контроль над организацией
  • Назначение администраторов и модераторов
  • Удаление организации
  • Все права администратора

Администратор:

  • Создание и удаление проектов
  • Создание, редактирование и удаление задач
  • Управление пользователями (кроме владельца)
  • Финальное подтверждение выполненных задач

Модератор:

  • Создание и редактирование задач
  • Назначение ответственных
  • Подтверждение выполнения задач

Сотрудник:

  • Просмотр назначенных проектов и задач
  • Принятие задач в работу
  • Отметка о выполнении с подтверждением

Проекты

Проекты — категории для группировки задач. Выбирайте проект в боковом меню.

  • Создание: Владелец и администраторы создают проекты
  • Дедлайн: Можно установить срок завершения проекта
  • Доступ: Настраивается в админ-панели для каждого сотрудника

Создание задачи

При создании задачи указываются:

  • Название: Краткое описание задачи
  • Описание: Подробности и комментарии
  • Ответственные: Один или несколько сотрудников (поиск по имени)
  • Срок: Дедлайн выполнения
  • Файлы: До 2 файлов (макс. 10 МБ каждый)
При назначении задачи сотруднику отправляется email-уведомление.

Статусы задач

  • Задача поставлена
    Новая задача ожидает принятия сотрудником
  • В работе
    Сотрудник принял задачу и работает над ней
  • Завершена
    Сотрудник выполнил, ожидает проверки руководителем
  • Готово
    Руководитель подтвердил выполнение, задача в архиве
Нажмите на статус задачи, чтобы изменить его.

Двойной контроль выполнения

Система двойной проверки гарантирует качество выполнения:

  • Шаг 1: Сотрудник завершает задачу, пишет отчёт и прикрепляет подтверждение
  • Шаг 2: Руководитель проверяет работу и подтверждает выполнение

Только после подтверждения руководителем задача переходит в статус «Готово».

Подтверждение выполнения

Когда сотрудник завершает задачу, он обязан:

  • Написать комментарий о выполненной работе
  • Прикрепить файл-подтверждение (фото результата, документ, скриншот)

Без комментария и файла завершить задачу невозможно.

Мои задачи

Кнопка «Мои задачи» показывает все задачи, где вы назначены ответственным:

  • Красный бейдж — количество активных задач
  • Нажмите на задачу, чтобы перейти к ней
  • Обновляется в реальном времени

Информация о задаче

Кнопка на карточке задачи открывает подробности:

  • Полное описание и прикреплённые файлы
  • Список ответственных сотрудников
  • История изменений статуса
  • Отчёт о выполнении с подтверждением

Панель администратора

Доступна владельцам и администраторам в разделе «Панель управления»:

  • Пользователи: Список всех участников организации
  • Роли: Назначение ролей сотрудникам
  • Доступ: Настройка доступа к проектам для каждого
  • Удаление: Исключение пользователей из организации

Панель управления

В разделе «Панель управления» доступно:

  • Админ панель: Управление пользователями (для владельцев и админов)
  • Тема: Переключение между светлой и тёмной темой

Мобильная версия

Приложение адаптировано для смартфонов:

  • Меню: Кнопка открывает боковое меню
  • Задачи: Колонки располагаются вертикально
  • Управление: Увеличенные кнопки для удобного нажатия
