# Настройка Email уведомлений (Бесплатно)

Вы просили сделать так, чтобы уведомления приходили на почту. Поскольку мы отказались от сложного бэкенда (Cloud Functions), я внедрил самое простое и популярное решение для отправки писем прямо из браузера — **EmailJS**.

В данный момент код работает в режиме **симуляции** (показывает alert), чтобы вы могли протестировать процесс создания задачи. Чтобы письма реально уходили, вам нужно выполнить несколько простых шагов (это бесплатно и не требует карты).

## Инструкция по настройке

1.  **Регистрация**:
    *   Зайдите на [https://www.emailjs.com/](https://www.emailjs.com/) и создайте бесплатный аккаунт ("Sign Up Free").

2.  **Создание Email Service**:
    *   В панели управления нажмите "Add New Service".
    *   Выберите "Gmail" (или другой ваш почтовый сервис).
    *   Нажмите "Connect Account" и авторизуйтесь.
    *   Нажмите "Create Service".
    *   Скопируйте **Service ID** (например, `service_x82...`).

3.  **Создание шаблона письма**:
    *   Перейдите в "Email Templates" -> "Create New Template".
    *   В теме письма напишите: `Новая задача: {{task_title}}`.
    *   В теле письма напишите что-то вроде:
        ```
        Здравствуйте, {{to_name}}!
        
        Вам назначена новая задача в проекте "{{project_name}}".
        
        Задача: {{task_title}}
        Срок: {{task_deadline}}
        
        Пожалуйста, приступите к выполнению.
        ```
    *   Нажмите "Save".
    *   Скопируйте **Template ID** (например, `template_v9...`).

4.  **Получение Public Key**:
    *   Перейдите в "Account" (иконка профиля) -> "General".
    *   Скопируйте **Public Key** (например, `user_Kj8...`).

5.  **Вставка ключей в код**:
    *   Откройте файл `script.js`.
    *   В самом верху (строки 12-14) найдите объект `emailConfig`.
    *   Замените значения на ваши:
        ```javascript
        const emailConfig = {
            serviceID: "service_...", // Ваш Service ID
            templateID: "template_...", // Ваш Template ID
            publicKey: "user_..." // Ваш Public Key
        };
        ```

После этого уведомления будут реально приходить на почту ответственного сотрудника!
