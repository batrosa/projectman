# ProjectMan - Система управления проектами

## 🎯 Новые возможности: Автоматический вход

Приложение теперь **автоматически запоминает** вашу учетную запись!

### ✨ Что изменилось?

1. **Автоматический вход**: После первого входа вы больше не вводите email и пароль при следующих посещениях
2. **Роль по умолчанию**: При регистрации роль всегда устанавливается как **Читатель** (`reader`); повышение до администратора выполняется вручную через Firestore

### 🔐 Как это работает?

#### Первый вход:
1. Зарегистрируйтесь, указав email и пароль
2. Аккаунту автоматически присваивается роль **Читатель**
3. Начните работу

#### Последующие посещения:
1. Откройте приложение - вход выполнится автоматически
2. Начинайте работу!

### 🚀 Технические детали

#### Firebase Authentication Persistence
- Использует `LOCAL` persistence
- Сессия сохраняется даже после закрытия браузера
- Автоматическое восстановление при перезагрузке страницы

#### Firestore Structure
```
users/{userId}
  ├─ email: string
  ├─ role: "admin" | "reader" (управляет правами доступа)
  └─ lastLogin: timestamp

projects/{projectId}
  ├─ name: string
  ├─ description: string
  └─ createdAt: timestamp

tasks/{taskId}
  ├─ projectId: string
  ├─ title: string
  ├─ assignee: string
  ├─ deadline: string
  ├─ status: "in-progress" | "done"
  └─ createdAt: timestamp
```

### 📋 Развертывание

#### 1. Обновите Firestore Rules
См. файл `FIRESTORE_DEPLOYMENT.md` для инструкций

#### 2. Разверните на Netlify
```bash
# Если используете Netlify CLI
netlify deploy --prod

# Или через веб-интерфейс Netlify
# Просто загрузите файлы через drag-and-drop
```

### 🔄 Выход из системы

Чтобы полностью выйти из аккаунта:
1. Нажмите кнопку **"Выйти"** в боковой панели
2. Вам нужно будет снова ввести email и пароль

### 🛡️ Безопасность

- Email/Password аутентификация через Firebase
- При регистрации роль всегда устанавливается как `reader`; роль `admin` назначается вручную в Firestore
- Правила безопасности ограничивают доступ к данным пользователя

### 📱 Поддержка устройств

- ✅ Desktop (Chrome, Firefox, Safari, Edge)
- ✅ Mobile (iOS Safari, Chrome Mobile)
- ✅ Tablet
- ✅ Offline-ready (с Firebase persistence)

### 🐛 Устранение неполадок

**Проблема**: Не могу войти автоматически
- Проверьте, что Firestore Rules развернуты
- Очистите кэш браузера и попробуйте снова
- Проверьте консоль браузера на наличие ошибок
